## Purpose

Phase 96 establishes the first fully structured governance foundation
covering:

Workers  
Permissions

This checkpoint marks the point where governance becomes part of the

## What This Checkpoint Locks

The system must never regress to a state where:

Workers can gain decision authority  
Permissions can transfer operator authority  
Capabilities imply autonomy  
Automation can self-authorize  
Governance invariants are unverifiable
