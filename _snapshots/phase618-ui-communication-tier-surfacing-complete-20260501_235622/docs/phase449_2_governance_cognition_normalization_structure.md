is structurally consistent even when intake shape varies.

• Stable interpretation surfaces
• Deterministic output shape
• Comparable governance exposure

• Traceable evidence attachment

Normalization must NOT:

• Change governance authority
• Decide outcomes
• Introduce runtime evaluation
• Rewrite intake structure
• Trigger execution

Invariant:

Normalization standardizes presentation,
not reasoning behavior.
